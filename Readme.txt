Instalar todos los paquetes necesarios mediante:

pip install -r requirements.txt

para ejecutar
python ethnicity.py [Nombre del directorio que contiene las imágenes]

ejemplo:
python ethnicity.py European-descendants

El programa creará dos carpetas con imágenes recortadas y giradas dentro de European-descendants/

El prograna creará un .csv con la identificación del grupo étnico, identificación de la foto, la etiqueta y las coordenadas de los puntos de interés de los rostros.
